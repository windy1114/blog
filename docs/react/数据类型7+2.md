## null , undefined
null和undefined都代表没有
null意料之中，一般都是开始不知道值，我们手动设置为null,后期再赋值操作。 null没有空间，一般最好用null作为初始的空值。 let num = null
undefined 意料之外 let num 创建一个变量没有赋值，默认值是undefined
