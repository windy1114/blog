##十进制互相转换为十六进制（八进制）##
```js
(int).toString(16); // converts int to hex, eg 12 => "C" <br/>
(int).toString(8);  // converts int to octal, eg. 12 => "14" <br/>
parseInt(string,16) // converts hex to int, eg. "FF" => 255 <br/>
parseInt(string,8) // converts octal to int, eg. "20" => 16
```
